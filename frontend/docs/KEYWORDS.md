## Updating content

When adding or editing copy:

1. Prefer **primary keywords** in the first paragraph or H2 of a section, and in meta/OG/title.
2. Use **secondary keywords** in bullets, feature names, and body text without forcing.
3. Keep phrasing natural; avoid repeating the same phrase in consecutive sentences.
4. After changing primary keyword usage, update this file and `docs/SEO.md` if needed.
